Пример: Свойства блока

Последовательно дополняя стиль внешнего контейнера и его внутреннего блока новыми свойствами, проследим за изменением отображения

1. Никаких лишних свойств
Внешний контейнер
Внутренний блок

2. Внешний контейнер: { width:300 }
Внешний контейнер
Внутренний блок

3. Внутренний блок: { margin:20; padding:10 }
Внешний контейнер
Внутренний блок

4. Внутренний блок:
 { margin:20; padding:10; border-color:orange; border-width:thick }
   Задан цвет и толщина границы, но граница не отображается, так как не задано отличное от умолчания значение свойства border-style
Внешний контейнер
Внутренний блок

5. Внутренний блок:
 { margin:20; padding:10; border-color:orange; border-width:thick; border-style:inset }
   Добавлено свойство border-style , и граница отображается
Внешний контейнер
Внутренний блок

6. То же задание параметров для внутреннего блока, но с использованием стенографического свойства border. Получилось короче
 { margin:20; padding:10; border:orange thick inset }
Внешний контейнер

Работа с практикумом:

Перед вами лабораторный практикум по Веб-дизайну. На вкладке "Старт" находятся ссылки на лабораторные работы, которые оформлены в стиле шестиугольников с написанной в центре основной темой, изучаемой в каждой лабораторной работе. Порядок работ идет построчно.

Всего в практикуме 7 лабораторных работ:

  1. Изучение тегов структуры и форматирования в HTML - документах.
  2. Теги для управления медиа-контентом в HTML - документах.
  3. Создание гиперссылок в HTML - документах.
  4. Основы каскадных листов стилей (CSS).
  5. Таблицы и фреймы в HTML - документах.
  6. Создание форм в HTML - документах.
  7. Стилизация веб - страницы с помощью CSS.

При открытии любой лабораторной работы вверху веб-страницы вы увидите навигационное меню по практикуму. Номер открытой в данный момент работы обозначен звездочкой. Кнопка "Старт" возвращает на стартовую страницу. Вкладка "Инфо" содержит в себе дополнительную информацию. В ней указан список всех тегов HTML5 с их кратким описанием, список глобальных атрибутов для всех HTML-тегов, правила разметки HTML-кода, список CSS-свойств.

В правой части веб-страницы находится содержание с ссылками на соответствующие главы в лабораторной работе.

После окончания теории в каждой лабораторной работе идет раздел "Практика", который является обязательным к выполнению. Быстрый переход к практике осуществляется с помощью содержания.

В корневой папке практикума вы найдете папку "!Notepad++Portable", с помощью этой программы намного удобнее верстать веб-страницы, используйте Notepad++ вместо стандартного блокнота.
